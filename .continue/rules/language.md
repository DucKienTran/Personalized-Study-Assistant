# Project Language

English is the official language of this project.

Write all new code in English.

This includes:

- UI text
- API response messages
- Log messages
- Comments
- Docstrings
- README updates
- TODO comments
- Variable names (for new code)
- Function names (for new code)
- Commit messages

Do not introduce new Vietnamese text unless explicitly requested.

When translating existing code:
- Preserve behavior.
- Do not rename database columns or API contracts unless explicitly instructed.
- If a translation is ambiguous, ask before changing it.
