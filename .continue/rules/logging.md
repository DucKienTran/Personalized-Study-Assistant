# Logging Style

Always write logs in English.

Use sentence case.

Do not use ALL CAPS.

Start with a verb.

Examples:

✓ Processing document.
✓ Generating quiz.
✓ Cache hit.
✓ Cache miss.
✓ Saving document.
✓ Upload completed.
✓ Failed to load user profile.

Avoid:

✗ PROCESSING DOCUMENT
✗ START GENERATE QUIZ
✗ LOADING...
✗ SUCCESS!!!
✗ ERROR!!!!!!!!
